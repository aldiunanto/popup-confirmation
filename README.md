# Popup Confirmation
A simple popup confirmation using jQuery

Usage
------------

1. Import `libs.js`
2. 

2. Create your 'confirmation' element.
```html
<div id="confirmation">
 	<div class="container">
 		<div class="text"></div>
 		<div class="actions">
 			<button class="btn default negative">cancel</button>
 			<button class="btn primary positive">ok</button>
 		</div>
 	</div>
</div>
```
